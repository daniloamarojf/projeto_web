1. Display: Block

Elementos com display: block ocupam a largura disponível, criando uma nova linha antes e depois do elemento.

Este é um bloco. Ele ocupa toda a largura disponível e cria uma quebra de linha.
Outro bloco, o que mostra que os blocos vão para a próxima linha.

2. Display: Inline

Elementos com o display: inline se alinham horizontalmente, respeitando apenas o seu conteúdo e não criam quebras de linha.

Este é um elemento inline. Outro elemento inline-block. Mais um elemento iniline-block
